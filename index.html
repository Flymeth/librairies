CPSer

Documentation

Installation

Copiez-collez le code ci-dessous au dessus de la balise </body>

Utilisation

La librairie va utiliser les différentes classes que vous attribuez à certain éléments pour savoir comment l'élément va oppérer.

Certaines classes ne peuvent être utilisé qu'une seule fois, tandis que pour d'autre, vous pouvez les mettres dans autant d'élément que vous le souhaitez.

  • Classes à n'utiliser qu'une seule fois:
    1. starter
    2. restarter
    3. clicker
    4. timer
  • Classes utilisable plusieurs fois:
    1. clicks
    2. clock
    3. result

Le comportement du jeu peut aussi être modifié !

Pour cela, ajoutez une balise <meta> dans le <head> de votre page.

Puis, configurez la comme ceci:

Voici la liste des options que vous pouvez mettre:

  • defaultTimerValue

Classes à n'utiliser qu'une seule fois
Classes propriétés
starter Lors du click, le système le system de cps va se lancer. Cela comprend le timer, ainsi que le fais que vous puissiez clicker sur le bouton du clicker.
restarter Ce bouton va servir a remettre le system à 0. Il va reset le timer, le nombre de cps et le resultat.
clicker C'est sur cet element que l'utilisateur va pouvoir clicker le plus de fois qu'il puisse.
timer Cet élément doit obligatoirement être du type <input>. Cet élement va pouvoir servir à l'utilisateur de changer le temps du timer.
Cette classe n'est pas obligatoire !
Classes pouvant être utilisé plusieurs fois
classes propriétés
clicks affiche le nombre de click que l'utilisateur a fais depuis le début du timer.
clock Affiche le temps restant
result Affiche le résultat (en click/seconde)
Paramètres du jeu
paramètres valeurs
defaultTimerValue La valeur par défaut du timer. A noter qu'elle doit être comprise entre la valeur minimal et maximal du timer !
minTimerValue La valeur minimal du timer.
maxTimerValue La valeur maximal du timer.
timerSteps La précision du timer.
Ex.
Si elle est de 1, le timer fera "5; 4; 3, ...",
Si elle est de 0.1, le timer fera "5.0; 4.9; 4.8; ..."
resultArrounds La précision du resultat.
Ex.
Si cette option est égal à 0.1, le résultat sera (par exemple): "4.6"

Tips !

<body> prendra la classe "cps-started" si le timer a commencé, et "cps-ended" si le timer est terminé !

A noter que si le jeu n'a pas commencé, <body> n'aura aucune classe.

Utiliser vos couleurs

Pour paramétrer vos propres couleurs, vous devez inclure dans le <head> de votre document une balise <meta> de la façons suivante:

Voici un tableau avec les propriétés que vous pouvez entrer ainsi que leur fonctions:

propriétés descriptions
elements le nom des élémets des balises (ex: section, div, h1...)
props les attributs des éléments (ex: class, href, id...).
tags les balises des éléments (ex: <, >, /)
texts les textes en dehors des balises
strings les chaînes de caractères dans les balises (ex: class="ceci est une chaine de caractère")
notes tous les commentaires (ex: <!-- ceci est un commentaire -->)
Vous pouvez également cibler les éléments avec leur classes (par exemple: .tags {color: blue}) dans un fichier .css !
Plus d'options
propriétés descriptions
propsIncludesEgals [boolean] inclure oui ou non les "=" dans la couleurs des attributs des balises
