Utilisation

La librairie va utiliser les différentes classes que vous attribuez à certain éléments pour savoir comment l'élément va oppérer.

Certaines classes ne peuvent être utilisé qu'une seule fois, tandis que pour d'autre, vous pouvez les mettres dans autant d'élément que vous le souhaitez.

  • Classes à n'utiliser qu'une seule fois:
    1. starter
    2. restarter
    3. clicker
    4. timer
  • Classes utilisable plusieurs fois:
    1. clicks
    2. clock
    3. result

Le comportement du jeu peut aussi être modifié !

Pour cela, ajoutez une balise <meta> dans le <head> de votre page.

Puis, configurez la comme ceci:

Voici la liste des options que vous pouvez mettre:

  • defaultTimerValue
  • minTimerValue
  • maxTimerValue
  • TimerSteps
  • resultArrounds
starter

Lors du click, le système le system de cps va se lancer. Cela comprend le timer, ainsi que le fais que vous puissiez clicker sur le bouton du clicker.

restarter

Ce bouton va servir a remettre le system à 0. Il va reset le timer, le nombre de cps et le resultat.

clicker

C'est sur cet element que l'utilisateur va pouvoir clicker le plus de fois qu'il puisse.

timer

Cet élément doit obligatoirement être du type <input>. Cet élement va pouvoir servir à l'utilisateur de changer le temps du timer.

Vous n'êtes pas obligé d'utiliser cette classe.

clicks

Si un élément possède cette classe, il affichera le nombre de click que l'utilisateur a fais depuis le début du timer.

clock

C'est sur cet/ces élément(s) que le temps restant sera afficher.

result

Quand le timer est finis, cet élément affichera le nombre de cps de l'utilisateur.

defaultTimerValue

Cette valeur est la valeur par défaut du timer. A noter qu'elle doit être comprise entre la valeur minimal et maximal du timer !

minTimerValue

La valeur minimal que le timer peut avoir

maxTimerValue

La valeur maximal que le timer peut avoir

TimerSteps

La précision du timer.

Ex:
Si elle est de 1, le timer fera "5; 4; 3, ...",
Si elle est de 0.1, le timer fera "5.0; 4.9; 4.8; ..."

resultArrounds

La précision du résultat.

Ex:
Si cette option est égal à 0.1, le résultat sera (par exemple): "4.6"

Tips !

<body> prendra la classe "cps-started" si le timer a commencé, et "cps-ended" si le timer est terminé !

A noter que si le jeu n'a pas commencé, <body> n'aura aucune classe.
